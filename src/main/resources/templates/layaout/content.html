Bienvenido al Sistema

Selecciona una opción del menú para comenzar.

Resumen de cada sección:

  • Inicio: Pantalla principal con acceso a todas las funciones.
  • Empleado: Lista de empleados registrados, con opciones de gestión.
  • Nuevo Empleado: Formulario para agregar nuevos empleados a la base de datos.
  • Contrato: Sección para gestionar contratos laborales.
  • Reportes: Área para generar informes y análisis sobre la empresa.
